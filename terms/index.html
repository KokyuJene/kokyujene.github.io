新 高級ジェネ™ 利用規約

最終更新日: 2025年9月21日
制定日: 2025年8月23日

第1章 Webサイト利用規約

第1条(適用範囲)

本規約は、「新 高級ジェネ™」(以下「当サーバー」)が運営する公式Webサイト「https://kokyujene.github.io/」(以下「当サイト」)の利用に関して適用されます。当サイトを利用されるすべての方(以下「利用者」)は、本規約に同意したものとみなされます。

第2条(サービス内容)

当サイトは、マインクラフトサーバー「新 高級ジェネ™」に関する以下の情報を提供します

･サーバーの基本情報および接続方法
･サーバールールおよび利用規約
･最新のお知らせ､更新情報
･コミュニティ情報

第3条(利用者の遵守事項)

利用者は以下の事項を遵守してください
1. 本規約および関連法令を遵守すること
2. サイトの正常な運営を妨害しないこと
3. 虚偽の情報を提供しないこと
4. 他の利用者に迷惑をかけないこと

第4条(禁止事項)

利用者は以下の行為を行ってはいけません

1. 法令または公序良俗に違反する行為
2. 当サイトの運営を妨害する行為
3. システムやセキュリティに対する不正アクセス
4. 著作権､商標権､肖像権等の侵害
5. 個人情報の不正取得､利用
6. 誹謗中傷､差別的表現､ヘイトスピーチ
7. 商業目的での無断利用
8. その他、運営が不適切と判断する行為

第5条(知的財産権)

1. 当サイトに掲載されるすべてのコンテンツ(文章、画像、ロゴ、デザイン等)の著作権は当サーバー運営に帰属します
2. 個人的な利用または新 高級ジェネ™の紹介目的での利用は許可されますが、改変や商用利用は禁止です
3. ロゴの改変や誤解を招く使用は一切認められません

第6条(免責事項)

1. 当サイトの情報の正確性､完全性､有用性について保証いたしません
2. 当サイトの利用により生じた損害について、運営は責任を負いません
3. 外部リンク先での問題について、運営は責任を負いません

第2章 マインクラフトサーバー利用規約

第7条(サーバーの目的)

新 高級ジェネ™は、マインクラフトを通じて楽しく平和に人々との交流を深め、プレイヤー同士が楽しさや喜びを感じ合えるコミュニティサーバーを目的として運営されています。

第8条(サーバー利用の基本原則)

1. 楽しさの共有 全プレイヤーが楽しめる環境作りを心がける
2. 平和な交流 争いごとを避け、建設的なコミュニケーションを図る
3. 相互尊重: 他のプレイヤーの意見､プレイスタイルを尊重する
4. フェアプレイ: 正当な方法でゲームを楽しむ

第9条(アカウント管理)

1. サーバーへの参加には有効なマインクラフトアカウントが必要です
2. アカウントの管理責任は利用者にあります
3. 複数アカウントでの不正行為は禁止です
4. 適切でないユーザーネームやスキンの使用は禁止です

第10条(禁止行為)

以下の行為は即座にBAN(利用禁止)の対象となる場合があります

迷惑行為

･チャットスパム､連続投稿
･他プレイヤーへの暴言､誹謗中傷､ハラスメント
･アイテムの窃盗､無断使用
･建築物の無断破壊､改変
･意図的なラグ発生行為
･その他プレイヤーの妨害行為

不正ツール､チート行為

･チートクライアント､ハックツールの使用
･連打ツール､マクロの使用
･オーバーエンチャント品の作成､使用
･バグ､グリッチの悪用(発見時は必ず報告してください)
･その他ゲームバランスを著しく損なう行為

商業､宣伝行為

･現金取引(RMT)
･他サーバー､コンテンツの宣伝
･過度な商業活動

第11条(PvP(プレイヤー対戦)ルール)

1. PvPは指定エリア以外では禁止です
2. トラブル発生時は運営への報告を優先し、私闘は禁止です
3. 許可されたPvPクライアント(Onix, Flarial, Latite等)の使用は認められます

第12条(取引､商売ルール)

サーバー内での取引は以下の条件下で許可されます

1. 禁止される取引
･ゲームバランスを著しく損なうもの
･特定プレイヤーを陥れる目的のもの
･借金システムを利用したもの

2. 宝くじ等のルール

･最低3つ以上の当たりを用意すること
･詐欺的な内容は禁止

第13条(4Dスキンの使用)

1. 過度に装飾的でPvPに支障をきたすスキンは禁止です
2. 荒らしを目的としたスキンは禁止です
3. 帽子や小物程度の装飾は許可されます

第14条(通報､問題解決)

1. ルール違反や問題を発見した場合は、速やかに運営に報告してください
2. プレイヤー間のトラブルは運営が仲裁します
3. 報復や私的制裁は禁止です

第3章 共通規定

第15条(制裁措置)

1. 警告: 初回の軽微な違反
2. 一時的利用停止: 重大な違反または繰り返しの違反
3. 永久利用禁止(BAN): 悪質な違反または改善の見込みがない場合
4. 制裁措置は運営の判断により決定され、事前通知なく実施される場合があります

第16条(個人情報保護)

1. 運営は必要最小限の情報のみを収集します
2. 収集した情報は適切に管理し、第三者に提供しません
3. プレイヤーは他者の個人情報を無断で収集､公開してはいけません

第17条(規約の変更)

1. 本規約は必要に応じて変更される場合があります
2. 重要な変更については事前にお知らせします
3. 変更後の継続利用により、新規約への同意とみなされます

第18条(準拠法､管轄裁判所)

本規約は日本法に準拠し、紛争が生じた場合は日本の裁判所を管轄裁判所とします。

第19条(お問い合わせ)

本規約に関するお問い合わせは、サーバー内または公式サイトの問い合わせフォームからご連絡ください。
